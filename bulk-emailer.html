📖 Setup Guide
1. Install dependencies:
npm install

2. Start the backend:
node server.js

3. Create a Gmail App Password:
Enable 2FA at myaccount.google.com, then go to myaccount.google.com/apppasswords and generate an App Password for "Mail".

4. Google Sheet column formats:
Senders sheet: email, password, name, daily_limit
Queue sheet: to_email, subject, body, cc, reply_to

5. Publish sheet as CSV:
File → Share → Publish to web → Select sheet → CSV → Publish. Use that URL in the import fields.
